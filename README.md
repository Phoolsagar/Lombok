# 🚀 Lombok Practice Projects

This repository contains multiple hands-on examples demonstrating the use of **Project Lombok** in Java.  
Each folder represents a separate example covering different Lombok annotations.

---

## 📂 Project Structure
- Lombok_Ex-1
- Lombok_Ex-2
- Lombok_Ex-3
- Lombok_Ex-4
- Lombok_Ex-5
- Lombok_Ex-6

Each example focuses on reducing boilerplate code using Lombok features such as:

- @Getter and @Setter  
- @ToString  
- @NoArgsConstructor and @AllArgsConstructor  
- @EqualsAndHashCode  
- @Data  
- @Builder  

---

## 🛠️ Technologies Used

- Java (JDK 21 LTS)  
- Project Lombok  
- Maven 
- STS
